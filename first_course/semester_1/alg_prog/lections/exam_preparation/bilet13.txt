1. Алгоритмы оптимального поиска
>> Оптимальные алгоритмы поиска предназначены для нахождения лучшего решения задачи в конечном простанстве решений. Они минимизируют (или максимизируют)
заданную целевую функцию при соблюдении ограничений.

# Примеры задачи оптимального поиска:
* Задача коммивояжера
* Задачи маршрутизации
* Задачи рзмещений и распределения
# Ключевые принципы:
* Разделение пространства решений на подмножества
* Вычисление оценок (верхних и нижних границ) для каждого подмножества
* Отсечение нерентабельных ветвей, чтобы избежать полного перебора всех вариантов

2. Метод ветвей и границ
>> метод ветвей и границ - это алгоритм для решения задач оптимизации, который последовательно делит пространство решений (ветвлений) и использует границы для отсечения частей, 
не содержащих оптимального решения.

# Основные этапы метода:
1. Ветвление: разбиение задачи на подзадачи, которые представляют собой подпространства исходной задачи
2. Оценка границ: вычисление нижней и верхней границы целевой функции для каждой подзадачи
3. Отсечение: исключение тех подзадач, у которых границы хуже текущего оптимального решения
4. Выбор подзадачи для дальнейшего ветвления: обычно выбирается подзадача с наилучшей оценкой

# Применение:
* Решение задачи коммивояжера, где нужно найти кратчайший маршрут, проходящий через заданные точки
